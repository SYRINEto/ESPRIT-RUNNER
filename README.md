# ESPRIT-RUNNER
 Un jeu de collecte d'objets développé entièrement en HTML, CSS et JavaScript, dont l'univers est directement inspiré de notre école ESPRIT.
 Description du jeu :
• Genre : Runner / Collecte d'objets
• 3 zones jouables : Hall d'entrée, Couloir des salles, Atelier Workshop
• Personnages : étudiant garçon ou fille en sweat ESPRIT
• Carte étudiant personnalisable avant de jouer (nom, prénom, classe, spécialité, photo)
• Obstacles : chaises, cartables, profs en patrouille
• Objets à collecter : livres, laptops, café, badges ESPRIT
• Guide intégré, système de score et 3 vies

Technologies utilisées :
• HTML5 / CSS3 / JavaScript vanilla
• Canvas API pour le rendu du jeu
• Aucune bibliothèque externe
• Poids total < 50 Mo
